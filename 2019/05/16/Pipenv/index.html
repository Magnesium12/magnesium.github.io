Pipenv

  • 一个独立且完整的 python 虚拟环境管理
  • 提供基于独立环境的独立 python 项目管理方法
  • 可以帮助我们摆脱多个项目的依赖库相互排斥的问题

为什么选择Pipenv

  1. python 社区主流推荐
  2. 自动记录第三方库依赖
  3. 采用哈希校验，包管理操作更加安全
  4. 可以根据 pipfile 和 pipfile.lock 快速创建配置虚拟环境
